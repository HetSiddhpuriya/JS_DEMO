# JavaScript Demo

A collection of JavaScript examples and exercises covering various concepts from basic to advanced.

## File Descriptions

### Root Directory

- [**`Adv_Inter.js`**](Adv_Inter.js): Advanced and intermediate JavaScript concepts, including variables, data types, operators, control flow, loops, functions, and arrays.
- [**`arr_exe2.js`**](arr_exe2.js): Array `map` method exercises with real-world examples.
- [**`arr_filter_exe.js`**](arr_filter_exe.js): Array `filter` method exercises with real-world examples.
- [**`arr_methods_exe.js`**](arr_methods_exe.js): A collection of 20 exercises covering various array methods like `map`, `filter`, `reduce`, `some`, `every`, `find`, etc.
- [**`arr_reduce_exe.js`**](arr_reduce_exe.js): Array `reduce` method exercises with real-world examples.
- [**`array_exe.js`**](array_exe.js): Exercises on basic array methods like `push`, `pop`, `shift`, `unshift`, `splice`, `slice`, `reverse`, and `sort`.
- [**`array.js`**](array.js): Explanation and examples of JavaScript array methods.
- [**`controlflow.js`**](controlflow.js): Examples of control flow statements like `if-else`, `switch`, and the early return pattern.
- [**`datatype.js`**](datatype.js): Simple examples of JavaScript data types and type coercion.
- [**`dom_task.js`**](dom_task.js): Questions and answers related to DOM manipulation, along with practical tasks.
- [**`dom.js`**](dom.js): Explanation and examples of DOM manipulation in JavaScript.
- [**`event.html`**](event.html): HTML file for demonstrating various DOM events.
- [**`event.js`**](event.js): JavaScript code for handling DOM events like `click`, `change`, `input`, `submit`, etc.
- [**`exercise.js`**](exercise.js): Basic JavaScript exercises on operators and conditional statements.
- [**`exercuse2.js`**](exercuse2.js): Exercises combining `filter`, `map`, and `reduce` for practical scenarios.
- [**`extra.js`**](extra.js): Comparison between `find` and `some` array methods with examples.
- [**`formValidation.html`**](formValidation.html): HTML structure for a modern form with input fields.
- [**`formValidation.js`**](formValidation.js): JavaScript for reading values from a form.
- [**`fun_exe.js`**](fun_exe.js): Exercises on functions, including declaration, expression, arrow functions, parameters, arguments, hoisting, closures, and IIFE.
- [**`functions.js`**](functions.js): Detailed explanation of JavaScript functions, including types, parameters, arguments, and advanced concepts.
- [**`index.html`**](index.html): The main HTML file that includes links to various JavaScript files.
- [**`interview.js`**](interview.js): A collection of 50+ interview questions and answers related to JavaScript array methods.
- [**`KeyDown.html`**](KeyDown.html): HTML file for demonstrating the `keydown` event.
- [**`KeyDown.js`**](KeyDown.js): JavaScript code to handle the `keydown` event.
- [**`loop_exe.js`**](loop_exe.js): Exercises on loops (`for` and `while`).
- [**`loops.js`**](loops.js): Examples of `while` and `do-while` loops, and `break` and `continue` statements.
- [**`mousemove.html`**](mousemove.html): HTML file for demonstrating the `mousemove` event.
- [**`mousemove.js`**](mousemove.js): JavaScript code to handle the `mousemove` event.
- [**`objects.js`**](objects.js): Explanation and examples of JavaScript objects, including creation, access, nesting, destructuring, and looping.
- [**`operators.js`**](operators.js): Examples of various JavaScript operators.
- [**`README.md`**](README.md): This file.
- [**`submit.html`**](submit.html): HTML file for a form that creates a card on submission.
- [**`submit.js`**](submit.js): JavaScript code to handle the form submission and create a card dynamically.

### `card/`

- [**`index.html`**](card/index.html): HTML structure for a note card stack application.
- [**`NoteCard.css`**](card/NoteCard.css): CSS for styling the note card stack application.
- [**`script.js`**](card/script.js): JavaScript for the note card stack application, including creating, updating, and deleting notes.

### `SearchBar/`

- [**`searchbar.html`**](SearchBar/searchbar.html): HTML structure for a search bar with a card container.
- [**`searchbar.js`**](SearchBar/searchbar.js): JavaScript for filtering and displaying user cards based on search input.

### `Storage/`

- [**`README.md`**](Storage/README.md): Explanation of the web storage and cookies demo.
- [**`storage.html`**](Storage/storage.html): HTML file for demonstrating web storage and cookies.
- [**`storage.js`**](Storage/storage.js): JavaScript code for demonstrating `localStorage`, `sessionStorage`, and cookies.

### `This/`

- [**`this.html`**](This/this.html): An empty HTML file.
- [**`this.js`**](This/this.js): Explanation of the `this` keyword in JavaScript.

### `Timmer/`

- [**`Timmer.html`**](Timmer/Timmer.html): HTML structure for a timer and a download progress bar.
- [**`Timmer.js`**](Timmer/Timmer.js): JavaScript code for `setTimeout`, `setInterval`, and creating a timer and progress bar.

### `ToDoList/`

- [**`todo.html`**](ToDoList/todo.html): HTML structure for a simple to-do list application.
- [**`todo.js`**](ToDoList/todo.js): JavaScript for adding, editing, and deleting tasks in the to-do list.
